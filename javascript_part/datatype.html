자바스크립트 자료형

자바스크립트 자료형은 다른 프로그래밍언어에 비해서 조금 유연하고, 자동으로 캐스팅 되는 특징이 있습니다.

  1. 정수(x) --> number
    1. 리터럴 방식 var level = 100;
    2. 객체 생성자 var level = new Number(100);
  2. 문자열 --> string
  3. 불리언 --> boolean
  4. null
  5. undefined : 변수 선언시 임의로 할당
  6. 배열 --> Array
  7. 함수 --> function
  8. 오브젝트 -> object
  9. 클래스 --> class (새롭게 등장, 이전에는 없어서 prototype 형태로 구현)
  10. 심볼 --> symbol (새롭게 등장)
