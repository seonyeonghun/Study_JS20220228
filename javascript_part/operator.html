연산자

프로그래밍을 위한 자바스크립트 연산자

종류

산술연산자

  1. + : 덧셈
  2. - : 뺄셈
  3. * : 곱셈
  4. / : 나눗셈
  5. % : 나머지

대입연산자

  • A = B : B항을 A항에 대입(=할당)

복합대입연산자(complex)

  • +=
  • -=
  • *=
  • /=
  • %=

증감연산자I

  • ++ : 1증가
  • -- : 1감소

증감연산자II

  • 전위 증감 연산자 : ++변수명 or --변수명
  • 후위 증감 연산자 : 변수명++ or 변수명--

비교 연산자

  • >
  • >=
  • <
  • <=
  • == : 값
  • === : 값, 자료형
  • != : 값
  • !== : 값, 자료형

논리연산자

  • NOT : !
  • AND : &&
  • OR : ||

삼항 조건 연산자

  • 조건 ? 코드1:코드2;

※복잡한 조건식/조건비교는 if조건문으로 해야합니다.

퀴즈! 연산자, 변수를 활용한 문제
